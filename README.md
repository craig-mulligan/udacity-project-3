Catalog project	
==============

flask crud excercise

## To run

```
git clone https://github.com/craig-mulligan/udacity-project-3.git
```

```
cd vagrant
```

```
vagrant up
```

```
vagrant ssh
```

```
cd /vagrant/catalog
```

```
python run.py
```

to shut down vagrant 

```
vagrant halt
```

Have fun!
